Python Password Safe-Check

This was a project I started when I first learned python.

The idea is that you can check whether your password was ever leaked and if so how many times.

The first concern is whether or not to trust your password to some api over the internet. But that's an issue that's solved.

Instead of sending your entire password, only a fraction of the HASHED password is sent over the wire.

The api used is from pwnedpasswords.

Technologies:
  • - Python
  • - hashlib
  • - requests
  • - sys
  • - pwnedpasswords.com
